## About The Project

[![Product Name Screen Shot][product-screenshot]](https://example.com)

Currently, several main OSes dominate the market. In my opinion, where all of these fall short is when it comes to the aspect of being a power-user. All of these operating systems have many restrictions - and while many of these are for security, which BambooOS will also preserve, many are to protect the user from themself. BambooOS serves as an operating system to provide the beauty of a modern operating system, while giving the user access to a much lower level, feature rich development experience.

### What does OSDev take?

Developing an operating system is a massive feat. It takes a deep understanding of the system architecture, ability to code in low level languages and capacity to learn quickly. While developing an OS, you gain an even deeper understanding of system internals and how modern computers function, and this is something I have experienced greatly while working on BambooOS, particularly due to writing for the x86_64 architecture.
